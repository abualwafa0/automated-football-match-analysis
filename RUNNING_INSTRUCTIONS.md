# دليل تشغيل مشروع تخرج 2

يتكون هذا المشروع من جزأين رئيسيين: واجهة المستخدم (Frontend) والخادم (Backend). اتبع الخطوات أدناه لتشغيل النظام بالكامل.

## 1. تشغيل واجهة المستخدم (Frontend - React/Vite)
يوجد كود الواجهة في مجلد `football-web-ai-analytics`.

### المتطلبات المسبقة:
- تثبيت [Node.js](https://nodejs.org/).

### خطوات التشغيل:
1. افتح موجه الأوامر (Terminal أو CMD) وانتقل إلى مجلد الواجهة:
   ```cmd
   cd "c:\Users\user\Desktop\مشروع تخرج 2\football-web-ai-analytics"
   ```
2. قم بتثبيت الحزم البرمجية:
   ```cmd
   npm install
   ```
3. قم بتشغيل خادم التطوير:
   ```cmd
   npm run dev
   ```
4. بعد التشغيل، يمكنك فتح المتصفح على الرابط الذي يظهر في الشاشة (مثل `http://localhost:3000` أو `http://localhost:5173`).

---

## 2. تشغيل الخادم (Backend - Python/FastAPI)
يوجد كود الخادم في مجلد `FootballAnalyticsBackend`.

### المتطلبات المسبقة:
- تثبيت [Python](https://www.python.org/).

### خطوات التشغيل:
1. افتح نافذة موجه أوامر (Terminal أو CMD) جديدة وانتقل إلى مجلد الخادم:
   ```cmd
   cd "c:\Users\user\Desktop\مشروع تخرج 2\FootballAnalyticsBackend"
   ```
2. قم بتفعيل البيئة الافتراضية الخاصة بالمشروع (مهم جداً للتعرف على uvicorn):
   ```cmd
   venv_tracking\Scripts\activate
   ```
3. تأكد من تثبيت الحزم (إذا لم تقم بذلك مسبقاً):
   ```cmd
   pip install -r requirements_tracking.txt
   ```
4. لتشغيل الخادم، استخدم الأمر التالي:
   ```cmd
   uvicorn backend.app:app --reload --host 0.0.0.0 --port 8000
   ```
   *(إذا لم يعمل الأمر السابق، يمكنك استخدام `python -m uvicorn backend.app:app --reload --host 0.0.0.0 --port 8000`)*
5. سيبدأ الخادم في العمل وسيكون جاهزاً للاتصال بواجهة المستخدم على المنفذ 8000.

---

## 3. كيفية نقل المشروع إلى جهاز آخر وتشغيله
لنقل المشروع إلى جهاز جديد (للدكتور أو لجنة المناقشة مثلاً) وتضمن أنه سيعمل بدون أخطاء، اتبع الخطوات التالية:

### الخطوة الأولى: التجهيز للنسخ (على جهازك الحالي)
لا تقم بنسخ المشروع وهو يحتوي على ملفات التثبيت الضخمة لأنها تعتمد على مسارات جهازك الحالي وقد لا تعمل على جهاز آخر.
1. يفضل حذف مجلد `node_modules` الموجود داخل `football-web-ai-analytics`.
2. يفضل حذف مجلد `venv_tracking` الموجود داخل `FootballAnalyticsBackend`.
3. قم بضغط المجلد الرئيسي `مشروع تخرج 2` بالكامل إلى ملف `.zip` أو ضعه على فلاشة (USB).

### الخطوة الثانية: التشغيل (على الجهاز الجديد)
يجب أن يحتوي الجهاز الجديد على الأقل على **Node.js** و **Python** مثبتين مسبقاً.

1. انقر بزر الماوس الأيمن على الملف المضغوط واختر "استخراج هنا" (Extract Here).
2. **لتشغيل الواجهة (Frontend):**
   - افتح موجه الأوامر في مجلد `football-web-ai-analytics`.
   - اكتب `npm install` (سيقوم بتنزيل الملفات الناقصة بناءً على الجهاز الجديد).
   - بعد الانتهاء، اكتب `npm run dev`.

3. **لتشغيل الخادم (Backend):**
   - افتح موجه أوامر جديد في مجلد `FootballAnalyticsBackend`.
   - قم بإنشاء بيئة افتراضية جديدة: `python -m venv venv_tracking`
   - قم بتفعيلها: `venv_tracking\Scripts\activate`
   - قم بتثبيت المكتبات: `pip install -r requirements_tracking.txt`
   - قم بتشغيل السيرفر: `uvicorn backend.app:app --reload --host 0.0.0.0 --port 8000`

بهذه الطريقة تضمن أن المشروع سيُبنى من الصفر على الجهاز الجديد وسيعمل بنسبة 100% دون أي أخطاء متعلقة بمسارات الملفات القديمة!
